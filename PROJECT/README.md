# Equipment Management System (PHP + Oracle)

Simple training equipment manager with user/admin flows.

## Setup
1. Update `config.php` with your Oracle credentials/DSN.
2. Run `schema.sql` in Oracle SQL Developer or the VSCode Oracle extension.
3. Place the project in your PHP server root (e.g., `htdocs/PROJECT`).
4. Access `http://localhost/PROJECT/`.

## Default admin
- Username: `admin`
- Password: `admin123`

## Features
- User registration/login, borrow, return, feedback.
- Admin equipment CRUD, dashboard, borrow history.
- Search/filter catalog, quantity tracking, flash alerts.
